# Proyecto_bsale

Tecnologias:
- python flask
- python sqlAlchemy
- bootstrap
- vanilla Javascript
- css

Como ejecutar backend:
- instalar python version 3.10
- instalar pipenv con comando pip install pipenv
- ejecutar comando pipenv install
- ejecutar comando pipenv shell
- ejecutar comando python app.py
